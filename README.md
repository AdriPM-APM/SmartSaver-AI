# SmartSaver-AI

SmartSaver-AI es una aplicación web impulsada por inteligencia artificial (IA) que genera cupones de descuento automatizados para tiendas. Este proyecto es experimental y se encuentra en fase inicial de desarrollo. 

## Estado del Proyecto

Este proyecto es **solo para uso personal** y no tiene la intención de ser lanzado al público. Actualmente, no es funcional y está destinado a pruebas y desarrollo personal.

## Descripción

La aplicación utiliza OpenRouter AI para generar cupones de descuento. Aunque el sistema está diseñado para facilitar la gestión de cupones, el autor recomienda no utilizarlo en su forma actual debido a su estado de desarrollo.

## Funcionalidades Principales

- Generación automática de cupones para tiendas mediante IA.
- Integración con OpenRouter AI para procesamiento de IA gratuito.
- Sistema de gestión de cupones con base de datos Prisma.
- Interfaz web para interacción con el sistema.

## Pila de Tecnología

- OpenRouter AI (para generación de IA)
- Prisma (ORM para gestión de base de datos)
- Node.js/JavaScript (inferido por la estructura del proyecto)
- Gestión de variables de entorno (.env)
