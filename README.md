# SmartSaver-AI

SmartSaver-AI es una aplicación web impulsada por inteligencia artificial (IA) que genera cupones de descuento automatizados para tiendas. Este proyecto es experimental, fue creado en toda parte con asistencia de IA y se encuentra en una fase inicial de desarrollo.

## Estado del Proyecto

Este proyecto es exclusivamente para uso personal, sin fines de lucro, y no tiene la intención de ser lanzado ni distribuido al público. En su estado actual no es funcional, se considera un entorno de pruebas y está destinado únicamente a aprendizaje y desarrollo personal. Es probable que el proyecto no alcance un grado de madurez suficiente para su uso real.

## Descripción

La aplicación utiliza OpenRouter AI para generar cupones de descuento. Aunque el sistema está diseñado para facilitar la gestión de cupones, el autor desaconseja su uso en su forma actual debido a su carácter experimental, a las limitaciones técnicas presentes y a la alta probabilidad de que el proyecto no evolucione hacia una solución estable.

## Funcionalidades Principales

- Generación automática de cupones para tiendas mediante IA.
- Integración con OpenRouter AI para procesamiento de IA gratuito.
- Sistema de gestión de cupones con base de datos Prisma.
- Interfaz web para la interacción con el sistema.

## Pila de Tecnología

- OpenRouter AI (para generación de IA)
- Prisma (ORM para la gestión de la base de datos)
- Node.js/JavaScript (inferido por la estructura del proyecto)
- Gestión de variables de entorno (.env)
